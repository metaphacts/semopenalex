What exactly do we provide?
  1. A semantic search interface to access, search and visualize the SemOpenAlex data in real time.
  2. Periodically updated RDF dump files of SemOpenAlex.
  3. URI Resolution of SemOpenAlex within the Linked Open Data Cloud.
  4. A publicly accessible SPARQL endpoint containing the latest SemOpenAlex data.
  5. Entity Embeddings for all works, authors, sources and institutions in SemOpenAlex.

Potential use cases
  • Entity-centric exploration of publications, authors, institutions, etc. (e.g., concerning some research area)
  • Easier data integration through use of RDF and by linking resources to other data sources (e.g., combining SemOpenAlex with digital library collections in RDF).
  • Data analysis and knowledge discovery (e.g., measuring the popularity of publications and authors; build semantic recommender systems for publications, authors, and sources; and analyzing the evolution of concepts, research methods and data sets over time).

Entity Embeddings
Trained DistMult entity embeddings for all works, authors, sources and institutions in SemOpenAlex.
For more details, please refer to GitHub.
URI Resolution
The resources of SemOpenAlex are resolvable via HTTP and content negotiation. In this way, the knowledge graph is part of the Linked Open Data cloud. Examples for URI resolution with curl: curl -H "Accept:text/n3" https://semopenalex.org/work/W4239696231 curl -H "Accept:text/plain" https://semopenalex.org/work/W4239696231

Changelog
  • SemOpenAlex v6.0.0 (Release Date 2025-02-11):
    • Removed all RDF-star statements for all skos:Concept linked to Works. The skos:Concept are now linked as plain RDF triples. E.g. <https://semopenalex.org/work/W4239696231> <https://semopenalex.org/ontology/hasConcept> <https://semopenalex.org/concept/C2129575> . Only Concepts above the threshold of 0.3 are included.
    • Added 7 million new Author entities.
    • Added 8 million new Work entities.
    • Added 4 thousand new Institution entities.
  • SemOpenAlex v5.0.0 (Release Date 2024-06-09):
    • Re-modeled the skos:Concept linked to Works as RDF-star satements including the score. E.g. << <https://semopenalex.org/work/W4239696231> <https://semopenalex.org/ontology/hasConcept> <https://semopenalex.org/concept/C2129575> >> <https://semopenalex.org/ontology/hasScore> "0.89"^^xsd:float .
    • Added new controlled Vocabularies: SemOpenAlex Topics, SemOpenAlex Keywords, Medical Subject Headings (MeSH) and Sustainable Development Goals Taxonomy (SDG) that Works are annotated with. All Vocabularies can be accessed in the Vocabularies overview page in the Assets drop-down in the header.
